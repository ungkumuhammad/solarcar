# Key Rules Summary — BWSC 2027 Challenger Class

Quick reference for race planning and strategy. Full detail in `bwsc-2027-regulations.md`.

---

## Hard Limits

| Rule | Value |
|---|---|
| Solar array area | max 6 m² |
| Battery energy | max 3 kWh |
| Daily driving window | 08:00 – 17:00 (08:30 Day 1) |
| Speed limit (NT) | 130 km/h |
| Speed limit (SA) | 110 km/h |
| Control stop duration | 30 min (mandatory) |
| Number of control stops | 9 |
| Night driving | Prohibited |
| Grid charging | Prohibited |
| Battery swap | Prohibited (time penalty if forced by breakdown) |

---

## Effective Driving Time (3-Day Race Plan)

```
Total driving window   = 3 days × 9 h          = 27.0 h
Control stop loss      = 9 stops × 0.5 h        =  4.5 h
Net effective drive    = 27.0 − 4.5             = 22.5 h

Required avg speed     = 3,022 km ÷ 22.5 h     ≈ 134 km/h
```

> 134 km/h exceeds the NT speed limit of 130 km/h and far exceeds SA's 110 km/h.  
> A 3-day finish requires the car to run **at the legal speed limit continuously** with no margin.  
> Realistic target: aim for **~120 km/h average cruise** and reassess with the energy model.

## Battery State at Start
- Batteries may be **fully charged** before race start
- This is the only grid/external charging opportunity
- All subsequent charging: solar array only
